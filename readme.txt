== Description ==

Simple games to aid the process of learning the Japanese language

The game is nearly entirely javascript driven.
Adding the tag [ganbatte-kana] into a post or page content will insert the game in the respective post or page.

Note that the game will not show/work properly if your computer does not have support for Japanese characters.

= Features =

* Version 1.0 - Hiragana and Katakana game
* Version 1.1 - Added more games: "numbers up to 90", "days of the week", "days of month", "seasons", "numbers obove 90"

= Bug Fixes =

* Version 1.1 - Url of plug-in in JavaScript in now NOT hardcoded.
* Version 1.2 - Re-organized plug-in directories structure, removing the "trunk" subdirectory from ganbatte download zip archive, hopefully remembered to include the "images" directory

== Installation ==

1. Upload `ganbatte` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[ganbatte-kana]` in your post or page content
